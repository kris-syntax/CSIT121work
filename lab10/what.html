What is it?

DCAT which stands for Data Catalog Vocabulary which is a W3C standard that was created
so that computer systems or software can exchange and/or use information between data catalogs.
It’s used in order to publish catalogs on the web and describe data. It also helps to aggregate
metadata from catalogs.

Metadata is used to describe or explain and information resource often referred as “data about data”
(oercommons). Metadata is used for websites, on my blog I create meta descriptions that describe the
article that I am writing about its used for discoverability, its almost like a sneak preview into what
the web content will be about. DCAT is similar in that it is used to describe datasets. DCAT is an RDF
vocabulary, RDF which stands for Resource Description Framework which is used to for describing
data or metadata.
